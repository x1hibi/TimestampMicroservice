Request Header Parser Microservice

Request Header Parser Microservice consists of get information from request header as ip address, lenguage and software.


API sintax:

[host]/request-header/api/whoami


Example Usage:

https://microservices-x1hibi.onrender.com/request-header/api/whoami


Example Output:

{"ipaddress":"::ffff:159.20.14.100","language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
