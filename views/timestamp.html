Timestamp Microservice

Timestamp Microservice consists of obtaining the date either in UTC format or in milliseconds and returning both dates in a JSON object.


API sintax:

[host]/timestamp/api/date?

date can be in UTC formart or in miliseconds.


Example wit UTC date:

Click in below link to convert UTC date:

host/timestamp/api/2015-12-25

Example Output:

{"unix":1451001600000,"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}



Example wit date in miliseconds:

Click in below link to convert date in miliseconds:

host/timestamp/api/10000

Example Output:

{"unix":10000,"utc":"Thu, 01 Jan 1970 00:00:10 GMT"}
