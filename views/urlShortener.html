URL Shortener Microservice

URL Shortener Microservice save a URL in mongo database and return a id with location from original url to create a short URL.


Insert a URL




API sintax:

[host]/api/shorturl/short_url?

short_url is a Int that represent the id in database, when enter in browser this url you will be redirect to original URL


Example of usage:


[host]/url-shortener/api/shorturl/0
